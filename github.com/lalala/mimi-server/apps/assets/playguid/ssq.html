1 投注

双色球投注区分为红色球号码区和蓝色球号码区，红色球号码区由1-33共三十三个号码组成，蓝色球号码区由1-16共十六个号码组成。

(1) 普通投注

投注时选择6个红色球号码和1个蓝色球号码组成一注进行单式投注，每注金额人民币2元。单注最高奖金可达1000万元。

(2) 胆拖投注

在33个红色球号码中，选择1-5个号码作为每注都有的胆码，再补充其他不同的红球号码（至少2个）作为拖码进行投注，蓝球无胆码拖码之分，蓝球至少选1个。

3 开奖

双色球由中福彩中心统一开奖，每周二、四、日21:30开奖，每周3期。双色球每期开奖时，在公证人员封存销售数据资料之后，并在其监督下通过摇奖器确定开奖号码。
