二十年老站喜迎双色球头奖

2017-06-10  亨通彩

5月14日晚，中国福利彩票双色球游戏进行第2017055期开奖。当期双色球红球号码为07、12、13、20、24、31，蓝球号码为05。

辽宁中出的一等奖出自沈阳福彩007号投注站，彩站位于皇姑区黄河南大街62号，彩票是一张6+4蓝号复式票，共计花费8元钱，中奖者除中得1注一等奖外，还中得3注二等奖，总奖金6,212,320元，这也是沈阳市今年中出的第7注双色球头奖。
